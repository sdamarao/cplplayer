Factoids

The title image is assigned in the course metadata record (courseware player tab, title image field).

The images are maintained in xml/production/working/common-c/images.

The file sizes:

  • The desktop image is 713 x 594 pixels.

  • The mobile image is 984 x 533 pixels.

The naming convention:

  • Begin with title_ then simple category with 2 digit number, e.g., title_name01.jpg (rule is preferred).

  • The mobile image ends with _mobile.png, e.g., name_mobile.png (rule is required).

  • The desktop matches the mobile name (omit the text _mobile) and the extention is jpg, e.g., name.jpg (rule is required).
